    Prefix

    • 该内容显示前缀

    InventoryName

    • 使用/CC命令的时候GUI中显示的名称

    InventorySize

    • 设置GUI大小(默认上限: 54)

    KnockBack

    • 当玩家没有对应的钥匙时，是否击退玩家

    Physical-Accepts-Virtual-Keys

    • 抽奖箱是否可以通过物理方式接受钥匙(不知道啥意思)

    Virtual-Accepts-Physical-Keys

    • 抽奖形式是否可以通过/cc set menu 的页面打开并抽奖

    Give-Virtual-Keys-When-Inventory-Full

    • 当给予钥匙的时候，玩家物品栏已满。则提供虚拟的钥匙。如果设置为 false 则 钥匙会掉落在地上

    Need-Key-Sound

    • 当玩家没有对应的钥匙时，发送的声音。

    QuadCrate

    • 使用原理图更具有仪式感的抽奖方式时候为其设置进行该抽奖的进行时间。
    • Timer - 设置时间

    DisabledWorlds

    • 禁用的世界(默认设置为：world_nether 也就是地狱)

    其他部分

    • 其他部分是设置Gui界面的玻璃设置的。当然你可以设置为其他的样式
