    命令与权限

    返回目录

    命令 权限 描述
    /key crazycrates.access 检查自身持有钥匙的数量。
    /key [player] crazycrates.admin 检查玩家拥有的钥匙数量。
    /cc crazycrates.menu 打开CC菜单。
    /cc help crazycrates.access 显示CC命令帮助页面。
    /cc additem <crate> <prize> crazycrates.admin 将游戏内的物品添加到抽奖箱中。
    /cc admin crazycrates.admin 打开管理员钥匙界面。
    /cc preview <crate> [player] crazycrates.preview 打开玩家的抽奖箱预览。
    /cc list crazycrates.admin 显示所有抽奖箱。
    /cc open <crate> crazycrates.open 为玩家打开抽奖箱(前提是要有对应的钥匙)
    /cc open <crate> [player] crazycrates.open.other 当玩家有钥匙时，为其打开抽奖箱
    /cc forceopen <crate> [player] crazycrates.admin 免费为玩家打开抽奖箱。
    /cc tp <location> crazycrates.admin 传送到抽奖箱的位置。
    /cc give <Physical/Virtual> <crate> [amount] [player] crazycrates.admin 给予玩家抽奖箱钥匙。
    /cc giveall <Physical/Virtual> <crate> [amount] crazycrates.admin 为所有在线玩家提供抽奖箱的钥匙。
    /cc take <Physical/Virtual> <crate> [amount] [player] crazycrates.admin 取走玩家身上的钥匙。
    /cc transfer <crate> <player> <amount> crazycrates.transfer 玩家钥匙的交易指令。
    /cc set <crate> crazycrates.admin 将查看的方块设置为抽奖箱。
    /cc set menu crazycrates.admin 设置正在查看的抽奖箱打开界面。
    /cc reload crazycrates.admin 重新加载插件。
    /cc convert crazycrates.admin 将支持抽奖箱插件的文件，转为抽奖箱插件的文件。
    /cc set1/set2 crazycrates.admin 为抽奖箱制作一个新的原理图。仅限 1.13+
    /cc save <schematic file name> crazycrates.admin 将新的原理图保存原理图文件夹中。仅限 1.13+
