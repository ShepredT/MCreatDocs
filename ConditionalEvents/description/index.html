    允许特定是事件添加不同的条件。如果这些条件满足，则将执行自定义的操作。可以使用该插件执行的操作的可能性是无穷无尽的。更简单的方式理解插件，这里有一些例子：

    如果玩家在某个坐标中按下按钮(或任意方块),你可以为玩家执行一些操作，例如执行命令，向他发送消息，获得一些药水效果以及其他更多的操作。

    example1:
    type: block_interact
    conditions:
    - '%block_x% == 20'
    - '%block_y% == 60'
    - '%block_z% == 20'
    - '%block_world% equals lobby'
    - '%block% equals STONE_BUTTON'
    - '%action_type% equals RIGHT_CLICK'
    actions:
      default:
      - 'message: &aYou''ve received $500!'
      - 'console_command: eco give %player% 500'
      - 'playsound: ENTITY_PLAYER_LEVELUP;10;2'

    example2:
    type: player_attack
    conditions:
    - '%victim% equals PLAYER'
    - '%item% equals DIAMOND_SWORD'
    - '%item_name% equals VIP Sword'
    - '%random_1-10% >= 8'
    actions:
      default:
      - 'message: &aYour diamond sword poison effect was activated!'
      - 'to_target: give_potion_effect: POISON;120;1'
      - 'to_target: message: &cYou were poisoned by &e%player%&c!'

      - 'kick: &cWhat are you trying to do?'

    example4:

    conditions:
    - '%block_world% equals spawn'
    actions:
      default:
      - 'cancel_event: true'
      - 'message: &cYou can''t break or place blocks on this world.'
      - 'playsound: BLOCK_NOTE_BLOCK_PLING;10;0.1'
    ignore_with_permission: conditionalevents.ignore.event4
